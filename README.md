# 🌐 Landing Page - Curso JavaScript

Projeto de uma **Landing Page responsiva** desenvolvido durante meus estudos de **HTML, CSS e JavaScript**.

---

## 📸 Preview

> Projeto focado em layout moderno, responsividade e organização de código.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (básico)
- Google Fonts

---

## 📂 Estrutura do projeto

```
📁 projeto
 ┣ 📁 assets
 ┃ ┣ 📁 css
 ┃ ┣ 📁 img
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┣ 📄 script.js
```

---

## 🎯 Funcionalidades

- ✅ Menu fixo com navegação por âncoras
- ✅ Layout responsivo (mobile + desktop)
- ✅ Grid layout com CSS Grid
- ✅ Galeria de imagens dinâmica
- ✅ Tabela responsiva
- ✅ Formulário de contato
- ✅ Menu mobile com botão toggle
- ✅ Efeitos visuais com hover

---

## 📱 Responsividade

O projeto foi desenvolvido com foco em **responsividade**, adaptando:

- Layout em grid para telas menores
